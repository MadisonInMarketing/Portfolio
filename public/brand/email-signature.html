How to use this signature

  1. Gmail: Settings → See all settings → Signature → Create new → click the source/HTML icon → paste the code below.
  2. Apple Mail: Preferences → Signatures → create a new one → open Mail's HTML editor or drag the signature.html file directly into the signature field.
  3. Outlook: File → Options → Mail → Signatures → New → paste into the editor (HTML mode).

Note: Update the mailto, Instagram, and LinkedIn URLs with your real links before pasting. Web font (Cormorant Garamond) will fall back to Georgia in most email clients — it will still look elegant.
